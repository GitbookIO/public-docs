# Build your first integration (Quickstart)

> **Tutorial.** Follow every step in order — at the end your integration will be installed and running in your own organization.

## Before you begin

You need a GitBook account, Node.js installed, and about twenty minutes. Familiarity with TypeScript helps but isn't required.

## Step 1: Install the GitBook CLI

Install the CLI globally and authenticate it against your GitBook account.

## Step 2: Scaffold your integration

Run the scaffold command and answer the prompts. You get a working integration project with a manifest and an entry script.

## Step 3: Run it locally

Start the dev flow and see your integration respond to events in a test space.

## Step 4: Publish to your organization

Publish the integration privately to your org so it appears in the integrations catalog.

## Step 5: Install and see it work 🎉

Install your integration on a space and trigger it. When you see it respond in the space, you've shipped your first integration. Next: the development guides for unfurls, interactive blocks, and webhooks.
